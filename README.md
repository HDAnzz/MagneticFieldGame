# MagneticFieldGame
众所周知，导线在切割磁感线时会产生电动势

如果如果该导体是闭合回路的一部分时，回路内会产生电流，这一段导体会受到安培力的作用。
## 玩法
使用按键**W**、**A**、**S**、**D**控制你的小棒！

**W**、**S**分别会让你的导体棒来到上方和下方的导轨处，**A**、**D**键将会切换你小棒收到的水平外力的大小。

三个轨道的右边每隔一定时间会随机生成电阻，让你所在的导轨形成环路。

同时还会有从右往左的磁场随机生成，当你的小棒处于环路且切割这些运动的磁场时就会收到安培力。

你所要做的就是**保持自己的位置**，不让自己**跨过左右的虚线**。

现在，祝你好运！
